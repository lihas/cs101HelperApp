Introduction

Use this app to quickly load grades from excel sheets into moodle forms.

Installation

  1. Download source from gihub (link)
  2. Open google chrome, go to settings->extensions
  3. Select developer mode
  4. Click load unpacked extension...
  5. browse for the folder where the source code is present and select it.

Usage

  1. Navigate to moodle.iitb.ac.in
  2. Navigate to the page which shows all the submissions
  3. At the bottom of this page ensure that quick grading is enabled
  4. Open chrome extension (it's icon will be present on the right of address bar)
  5. Enter roll number and grades as shown in the following figure.
